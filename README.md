## Features
* Tooltips (дополнительная информация при наведении)
* Доступ к модальным окнам через state manager Vuex
* Гибкая таблица
* Организован роутинг страниц с помощью Vue-router (при клике на "Мои программы" открывается "/MyPrograms")
